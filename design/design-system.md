# Дизайн-система — Монетный анализатор

## Философия

Визуальный язык отсылает к нумизматическим каталогам и аукционным домам: строгость, патина старины, золото как акцент достоинства. Никакой цифровой «пластиковости» — только матовые поверхности, засечные шрифты, тонкие линии.

## Цветовая палитра

### Основные цвета

| Токен | HEX | Применение |
|---|---|---|
| `--bg` | `#152a15` | Фон страницы (British Racing Green) |
| `--bg-card` | `#1e3a1e` | Фон карточек, секций |
| `--bg-card2` | `#243d24` | Вложенные блоки, инпуты |
| `--text` | `#F5F0E8` | Основной текст (кремовый) |
| `--gold` | `#C9A84C` | Акцент: заголовки, кнопки, иконки |
| `--gold-dark` | `#a8852e` | Hover-состояние золотых элементов |
| `--muted` | `#7a9a7a` | Вторичный текст, подписи, плейсхолдеры |
| `--border` | `#2e5030` | Разделители, рамки карточек |
| `--danger` | `#c97a4c` | Ошибки, критические дефекты монеты |

### Правила использования цвета
- Золото `#C9A84C` — только для смысловых акцентов. Не использовать на больших площадях.
- Мутный зелёный `#7a9a7a` — весь вспомогательный текст, лейблы полей, подписи.
- Никогда не класть светлый текст на `--gold` фон без проверки контраста (WCAG AA).
- Фоновые слои идут строго от тёмного к светлому: `--bg` → `--bg-card` → `--bg-card2`.

## Типографика

### Шрифт

**Georgia** (системный засечный шрифт) — единственный шрифт проекта.

```css
font-family: Georgia, 'Times New Roman', serif;
```

Причина выбора: доступен везде без загрузки, передаёт нужную «музейную» серьёзность.

### Шкала размеров

| Использование | Размер | Особенности |
|---|---|---|
| H1 (Hero) | `clamp(2rem, 5vw, 3.2rem)` | font-weight: normal |
| H2 (секции) | `1.25rem` | font-weight: normal |
| Лейблы полей | `0.75rem` | letter-spacing: 0.15em, uppercase |
| Основной текст | `0.92–1.05rem` | line-height: 1.6–1.7 |
| Подписи, мелочь | `0.72–0.83rem` | color: var(--muted) |
| Цена / грейд | `1.5–1.6rem` | color: var(--gold) |

### Принципы
- Декоративных начертаний (`bold`, `heavy`) избегать — всё через размер и цвет.
- `font-style: italic` для цитат, исторических справок, подзаголовков.
- `letter-spacing: 0.15–0.25em` + `text-transform: uppercase` для лейблов и кнопок.

## Компоненты

### Кнопка — первичная (золотая заливка)

```css
background: #C9A84C;
color: #0e1e0e;
border: none;
font-family: Georgia, serif;
letter-spacing: 0.12em;
text-transform: uppercase;
padding: 16px 32px;
```
Hover: `background: #dbb95c`
Использование: главное CTA («Определить монету»).

### Кнопка — вторичная (золотая рамка)

```css
background: transparent;
border: 1.5px solid #C9A84C;
color: #C9A84C;
```
Hover: заливка золотом, текст тёмный.
Использование: «Отправить запрос», вспомогательные действия.

### Кнопка переключения языка

```css
border: 1px solid #C9A84C;
color: #C9A84C;
padding: 7px 18px;
font-size: 0.85rem;
```

### Карточка / Info Block

```css
background: #243d24;
border: 1px solid #2e5030;
padding: 18px 20px;
border-radius: 2px;  /* минимальное скругление */
```
Акцентированная карточка (результат): `border: 1px solid #C9A84C`

### Загрузочная зона

```css
border: 1.5px dashed #C9A84C;
background: #1e3a1e;
padding: 56px 32px;
```
Drag-over состояние: `background: #243d24; border-color: #e0bc6a`

### Разделитель секций

Горизонтальная линия с текстом по центру:
```
────────── ЗАГОЛОВОК СЕКЦИИ ──────────
```
Цвет линии: `#2e5030`, текст: `#C9A84C`, 0.72rem, uppercase, letter-spacing: 0.22em.

### Тег дефекта

```css
/* Значительный дефект */
background: rgba(201,151,76,0.1);
border: 1px solid rgba(201,151,76,0.3);
color: #C9A84C;

/* Незначительный дефект */
background: rgba(122,154,122,0.1);
border: 1px solid rgba(122,154,122,0.3);
color: #7a9a7a;
```

### Бейдж уверенности

```css

border: 1px solid #C9A84C;
color: #C9A84C;
padding: 5px 14px;
```

## Шкала грейдов

Стандарт Шелдона (Sheldon Scale), адаптированный для нумизматики.

| Аббревиатура | Название | Описание |
|---|---|---|
| Poor (P-1) | Плохое | Монета едва узнаваема, сильный износ |
| Fair (F-2) | Удовлетворительное | Тип определяется, детали почти стёрты |
| AG (AG-3) | Почти плохое | Основные детали видны |
| G (G-4/6) | Хорошее | Все детали видны, но плоские |
| VG (VG-8/10) | Очень хорошее | Детали чёткие, слабый рельеф |
| F (F-12/15) | Fine | Умеренный износ, все надписи чёткие |
| **VF (VF-20/30/35)** | **Very Fine** | **Лёгкий износ на выступающих деталях** |
| EF/XF (EF-40/45) | Extremely Fine | Минимальный износ, рельеф чёткий |
| AU (AU-50/55/58) | About Uncirculated | Следы обращения только на высоких точках |
| UNC (MS-60…70) | Uncirculated | Не обращалась, блеск чеканки сохранён |

### Визуальная шкала (HTML/CSS)

Трек прогресса шириной 100%, маркер на позиции текущего грейда:
- Poor = 5%, Fair = 14%, AG = 22%, G = 30%, VG = 38%
- F = 46%, **VF = 55%**, EF = 64%, AU = 75%, UNC = 95%

```css
.grade-fill {
  background: linear-gradient(90deg, #7a9a7a, #C9A84C);
  height: 6px;
  border-radius: 3px;
}
.grade-marker {
  width: 16px; height: 16px;
  background: #C9A84C;
  border-radius: 50%;
  border: 2px solid #152a15;
}
```

## Сетка и отступы

- Контентная ширина: `max-width: 820px`, центрирование `margin: 0 auto`
- Горизонтальные паддинги: `48px` (desktop) / `20px` (mobile ≤600px)
- Вертикальные отступы между секциями: `52–80px`
- Grid для карточек результата: `grid-template-columns: 1fr 1fr`, gap `20px`
- Grid для формы: `1fr 1fr`, full-width через `grid-column: 1 / -1`

## Анимации

- `fadeUp`: появление карточки результата — `opacity 0→1, translateY 16px→0`, 0.4s ease
- Все hover-переходы: `transition: 0.2s`
- Спиннер загрузки: `border-radius: 50%`, `animation: spin 0.7s linear infinite`
- Шкала грейдов: `transition: width 0.8s ease` при появлении карточки
