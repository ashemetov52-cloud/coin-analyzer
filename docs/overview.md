# Монетный анализатор — Обзор проекта

## Цель

Веб-сервис для автоматической идентификации монет по фотографии: определение страны, года, номинала, грейда (состояния) и рыночной стоимости. Финальная цель — снизить порог входа в нумизматику и дать коллекционерам быстрый первичный ориентир по цене без обращения к форумам или аукционам.

---

## Целевая аудитория

| Сегмент | Описание |
|---|---|
| Начинающие коллекционеры | Хотят понять, что досталось от бабушки или куплено на блошином рынке |
| Опытные нумизматы | Нужна быстрая вторая оценка или сортировка больших партий |
| Продавцы (Авито, eBay) | Хотят правильно выставить цену перед продажей |
| Эксперты Златника / НМС | Получают предварительно обработанные заявки через форму на сайте |

---

## Ключевые функции

### MVP (реализован в index.html)
- Загрузка фото монеты (drag-and-drop или клик)
- Демо-карточка результата: грейд, цена, история, дефекты, шкала грейдов
- Форма заявки на экспертную оценку (с согласием по ФЗ-152)
- Двуязычный интерфейс RU / EN

### Следующий этап (Claude Vision API)
- Реальная идентификация монеты через Vision API
- Возврат структурированного JSON: название, год, грейд, диапазон цен, дефекты
- Хранение истории запросов пользователя
- Экспорт результата в PDF

### Перспектива
- Мобильное приложение (PWA или React Native)
- База данных монет с ценовой историей
- Интеграция с аукционными площадками (Конрос, eBay, NumisMaster)

## Метрики успеха

### Продуктовые
| Метрика | Цель (3 мес) |
|---|---|
| Точность идентификации монеты | ≥ 85% |
| Точность грейда (±1 ступень от эксперта) | ≥ 75% |
| Время ответа API | < 4 сек |

### Пользовательские
| Метрика | Цель |
|---|---|
| Конверсия загрузка → анализ | ≥ 60% |
| Конверсия анализ → заявка эксперту | ≥ 8% |
| Возврат пользователя (retention 7d) | ≥ 25% |

### Технические
- Uptime: 99.5%
- Core Web Vitals: LCP < 2.5 с, CLS < 0.1
- Размер страницы (без изображений): < 150 КБ
